Introdução¶

Visão Geral¶

Este guia de estilo foi criado para os desenvolvedores Front-end da Jussi, mas esperamos que seja útil para outras equipes também, como de UX e Criação.

Poderá servir como guia de estilo visual para futuras criações de Landing Pages e etc, assim como padronizar os códigos responsáveis por exibir os elementos na interface do usuário.

Benefícios¶

- Podemos construir de forma consistente, concentrando nossa energia em fluxos de trabalho e lógica, não em formulários da Web e itens de lista.

- Podemos reutilizar o código em vez de reinventar a roda.

- Podemos ver todos os nossos padrões em um só lugar, resolvendo rapidamente problemas de manutenção.
